Objetivos

Esta segunda entrada concentra-se na aplicação de CSS3 para transformar a estrutura HTML da Entrada I em uma interface visual profissional, responsiva e acessível. Os alunos devem demonstrar domínio de leis modernas, sistemas de design e técnicas de estilização.

Especificações Técnicas Obrigatórias

•
Sistema de Design
Envolvedor de desenvolvimento sistema de projeto consistente com variáveis CSS personalizadas;
Definir paleta de núcleos com no mínimo 8 núcleos (primárias, secundárias, neutras);
Estabelecer tipografia hierárquica com no mínimo 5 tamanhos de fonte;
Criar sistema de espaçamento modular (8px, 16px, 24px, 32px, 48px, 64px).

•
Leiautes Responsivos com Caixa flexível e Grade
Implementar leiaute principal usando CSS grade para estrutura geral;
Utilizar caixa flexível para componentes internos e alinhamentos;
Criar no mínimo 5 pontos de interrupção responsivos bem definidos;
Sistema de desenvolvimento de grade personalizado (12 colunas);
Implementar leituras específicas para diferentes tipos de conteúdo.

•
Navegação Sofisticada e Interativa
Criar menu principal responsivo com submenu menu suspenso;
Implementar navegação móvel com menu hambúrguer.

•
Componentes da Interface
Sistema de desenvolvimento de cartas responsivos para projetos;
Criar botões com estados visuais (pairar, focar, ativo, desativado);
Implementar formulários padronizados com validação visual;
Desenvolver componentes de feedback (alertas, brindes, modais);
Criar sistema de emblemas e tags para categorização.

CSS Organizados
Estrutura de massas organizada (pastas, HTML, imagens e CSS);
Estrutura modular de CSS.
